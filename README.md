# IDE-AI-PRO
Software de programação em processo de desenvolvimento para qualquer programador usar
🚀 Sobre o Projeto

Este é um software de programação desenvolvido com o objetivo de oferecer uma plataforma acessível, flexível e poderosa para desenvolvedores de todos os níveis. O projeto é open source e está disponível para que qualquer pessoa possa utilizar, estudar, modificar e contribuir.

Atualmente, o sistema encontra-se em fase de desenvolvimento ativo, o que significa que algumas funcionalidades ainda estão em aprimoramento e podem existir erros ou comportamentos inesperados. Ainda assim, grande parte da estrutura principal já está implementada e funcional, permitindo testes reais e uso prático da ferramenta.

🧠 Funcionalidades Principais

O software foi projetado para integrar diferentes formas de desenvolvimento assistido por inteligência artificial, incluindo:

Suporte a modelos de IA offline, permitindo uso mesmo sem conexão com a internet
Integração com APIs externas, como:
OpenAI (ChatGPT)
Claude (Anthropic)
Ambiente voltado para produtividade em programação
Estrutura preparada para expansão com novas funcionalidades e automações

Além disso, o sistema foi pensado para funcionar de forma híbrida, combinando inteligência online e offline, oferecendo maior controle e autonomia ao usuário.

⚠️ Estado Atual

O projeto já possui uma base sólida, porém ainda requer evolução contínua. Algumas partes podem não estar totalmente otimizadas ou finalizadas. Por isso:

Este repositório representa a versão atual do sistema, sendo disponibilizado exatamente como está, para testes, validação e contribuição da comunidade.
