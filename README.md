# AI Engineering System

AI-powered IDE with autonomous agent, RAG, offline AI (Ollama), and more.

## Quick Start

### 1. Clone and install

```bash
git clone https://github.com/your-user/ai-engineering-system.git
cd ai-engineering-system
npm install
npm run build
```

### 2. Configure API keys

```bash
cp .env.example .env
```

Edit `.env` and add your API keys:
```
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
```

### 3. Run the desktop IDE

```bash
cd desktop
npm install
node_modules/.bin/electron.cmd main.js   # Windows
# or
npx electron main.js                      # Mac/Linux
```

## Offline AI (Ollama)

1. Install Ollama from [ollama.ai](https://ollama.ai)
2. Pull a model: `ollama pull qwen2.5-coder:7b`
3. Click **🔌 Offline: OFF** in the IDE toolbar

The IDE auto-detects Ollama from common install locations.
To use a custom path, set in `.env`:
```
OLLAMA_EXE_PATH=C:\path\to\ollama.exe
OLLAMA_MODELS=C:\path\to\.ollama\models
```

## Project structure

```
src/           TypeScript source
dist/          Compiled output (generated by npm run build)
desktop/       Electron desktop app
  main.js      Main process
  renderer.js  UI logic
  index.html   Interface
  extensions/  Drop .js extension files here
workspace/     Safe working directory for AI file operations
```
